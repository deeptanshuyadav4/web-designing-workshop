Web Designing Workshop – Course Curriculum

Subject Code: 25VA151

Academic Year 2025–26

This course introduces students to the fundamentals of web design. Topics include HTML, CSS, and basic JavaScript. Students will build real-world projects to strengthen their skills.

The curriculum is divided into eleven experiments, each targeting a specific set of concepts. Special characters: © 2026  •  & <HTML>

Syllabus Overview

Unit Topics (Ordered List)

  1. Introduction to HTML – structure, tags, attributes
  2. Text Formatting – headings, paragraphs, bold, italic
  3. Lists – ordered, unordered, description
  4. Links and Multimedia
  5. Tables
  6. Forms and Input Elements
  7. CSS Basics – selectors, box model
  8. CSS Layout – flexbox, grid
  9. Responsive Design – media queries

Tools Required (Unordered List)

  • VS Code (code editor)
  • Google Chrome (browser)
  • Git (version control)
  • GitHub (remote repository)

Key Terms (Description List)

HTML
HyperText Markup Language – the structure of a web page.
CSS
Cascading Style Sheets – controls the visual presentation.
JavaScript
A scripting language that adds interactivity to web pages.

Experiment Breakdown (Nested List)

  • HTML Experiments
    1. Course Curriculum Page
    2. News Website
    3. College Timetable
    4. Hotel Booking Form
    5. Student Registration Form
  • CSS Experiments
    1. E-Commerce Product Page
    2. Navigation Menu
    3. Student Profile Cards
    4. Responsive Product Grid
    5. E-Commerce Navbar
    6. Hamburger Navbar
