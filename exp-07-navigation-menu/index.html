display: block

Each element starts on a new line and stretches full width. Used for div, p, h1–h6 by default. Ideal for vertical menus.

display: inline

Element flows with surrounding text. Cannot set width/height. Used for span, a, strong by default.

display: inline-block

Flows inline like text, but respects width, height, and padding. Great for nav buttons side by side.

float & clear

float: left or float: right removes the element from normal flow and pushes it to the side. clear: both (or the clearfix hack) prevents subsequent elements from wrapping around the float.
