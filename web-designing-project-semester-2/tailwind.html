Tailwind Concepts Used on This Page

Mobile-First Breakpoints

sm: ≥640px  ·  md: ≥768px  ·  lg: ≥1024px

Base classes target mobile. Prefixed classes override at larger sizes.

Utility Classes

Single-purpose classes like flex, p-4, text-white, rounded-xl replace custom CSS.

Responsive Grid

grid-cols-1 → md:grid-cols-2 → lg:grid-cols-3 creates the 1/2/3 column layout.

Flexbox Utilities

flex-col on mobile → md:flex-row on desktop stacks header vertically then horizontally.
