How the Code Works

1. class & constructor

The Member class has a constructor(name, age, plan) that initialises properties when an object is created with new Member(...).

2. Inheritance (extends)

PremiumMember extends Member inherits all properties and methods of Member and adds its own (ptSessions, bookPT()).

3. super()

super(name, age, plan) inside the child constructor calls the parent constructor so inherited properties are set up correctly.

4. Method Overriding

PremiumMember and EliteMember both define their own getInfo() which replaces (overrides) the parent version.

5. Encapsulation (#private)

#joinDate is a private field — it cannot be accessed from outside the class. Only exposed via getJoinDate().

6. instanceof

instanceof checks whether an object was created from a specific class (e.g. elite instanceof Member → true because of inheritance).
