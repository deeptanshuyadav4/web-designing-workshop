# Web Designing Workshop (25VA151)

A collection of beginner-level web designing lab experiments completed as part of the **Web Designing Workshop** course (Subject Code: **25VA151**).

## Experiments

| # | Experiment | Topics Covered |
|---|------------|---------------|
| 01 | Course Curriculum Page | Headings, paragraphs, text formatting, all list types |
| 02 | News Website | Hyperlinks, navigation, embedded video, multimedia |
| 03 | College Timetable | Tables, rowspan, colspan, nested table, caption |
| 04 | Hotel Booking Form | Input fields, labels, select, textarea, buttons |
| 05 | Student Registration Form | Form validation, email, required attributes |
| 06 | E-Commerce Product Page | CSS box model, text, colors, fonts |
| 07 | Navigation Menu | Display block/inline/inline-block, float, clear |
| 08 | Student Profile Cards | Flexbox layout |
| 09 | Responsive Product Grid | CSS Grid, media queries |
| 10 | E-Commerce Navbar | Responsive navbar with CSS |
| 11 | Hamburger Navbar | Hamburger menu, checkbox hack, mobile-first |
