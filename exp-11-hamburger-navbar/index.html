How the Hamburger Menu Works (Checkbox Hack)

  1. A hidden <input type="checkbox"> stores open/closed state — no JS needed.
  2. The <label for="menu-toggle"> contains the three bar <span>s and acts as the clickable button.
  3. On mobile (@media max-width: 768px), the .nav-links is collapsed with max-height: 0; overflow: hidden.
  4. When the checkbox is checked, the sibling selector #menu-toggle:checked ~ .nav-links sets max-height: 400px, smoothly expanding the menu via CSS transition.
  5. The three bars animate into an ✕ using transform: rotate() and opacity: 0 on the middle bar.
